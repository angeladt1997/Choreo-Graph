New User User Story

After clicking "New User", user will be directed to this login page. User will enter an email and create a new username and a qualifying password which will be stored on the Choreo-Graph database. Users can also choose what state they reside in which will automatically populate a list of organizations in that state that use the software. Users can indicate that they already have an account which will direct them the the regular login page. Or, users can create their account using their qualifying information ny clicking create account, which will automatically log them in and push them to the dashboard.
